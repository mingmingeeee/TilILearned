Exception 관련 서술형 문제 출제

1. ArithmeticException: 정수를 0으로 나눈 경우 
2. FileNotFoundExceiption: 지정 경로에 파일을 참조할 수 없을 때 예외 발생
3. ArrayIndexOufOfBoundsException: 배열의 범위를 벗어난 Index를 참조할 때 예외 발생
4. NullPointerException: Null인 Object를 참조할 때 예외 발생
5. ClassNotFoundException: Class가 로드되지 않을 경우 예외 발생
6. IOException: IO Streaming 과정중에 발생하는 예외
